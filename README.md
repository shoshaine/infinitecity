An infinite 3D city that grows based on the mock market cap of the $CITY token.

## Features
- **Dynamic Growth**: The city expands and buildings get taller as the market cap increases.
- **Mock Market Cap**: Simulates a volatile token market cap from 3.45k to 16.2k.
- **Aesthetic Visuals**: Night-time city vibe with glowing windows, parks, and smooth animations.
- **Interactive Camera**: Orbit, zoom, and pan to explore the growing metropolis.

## Tech Stack
- **Vite**: Build tool
- **React**: UI Framework
- **Three.js / React Three Fiber**: 3D Rendering
- **React Three Drei**: 3D Helpers
- **Tailwind CSS**: Styling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`
